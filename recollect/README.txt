RECOLLECT — VERSION 1

Put these three files directly in your recollect folder:
  index.html
  style.css
  script.js

Open index.html in a browser or deploy this folder as a static site.

Included:
- Deck library with deck/tag search
- Deck title, tags, and card count
- Text and/or image on either card side
- Image source-file limit of 5 MB
- Interactive square crop preview with drag and zoom
- Flashcard study in either direction
- Whole-deck matching with scrollable columns
- Local browser storage (no account or backend)

Important:
Your decks are saved in the browser/device where you create them. Browser storage is local only for Version 1, so data will not automatically sync between devices.
